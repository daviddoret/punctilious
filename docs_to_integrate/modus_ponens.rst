# Modus ponens

## Definition

_modus_ponens is a well-known _inference-rule.

## Formal definition

(P ⟹ Q), P ⊢ Q

## Synonyms

- Affirming the antecedent
- Implication elimination
- Modus ponendo ponens
- Modus ponens
- MP
