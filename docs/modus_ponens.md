# Modus ponens

## Definition

_Modus ponens_ is a well-known _inference rule_.

## Formal definition

(P ⟹ Q), P ⊢ Q

## Synonyms

- Affirming the antecedent
- Implication elimination
- Modus ponendo ponens
- Modus ponens
- MP
